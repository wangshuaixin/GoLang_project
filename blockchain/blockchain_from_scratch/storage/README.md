### 过程

1. 打开一个数据库（DB）文件

2. 检查数据库文件是否已有其他区块链文件存储，是或者否分两种情况不同论述

3. 如果已经有了一个区块链：// Get "l"

    1. 创建一个新的 Blockchain 实例 

    2. 将新建 Blockchain 实例的顶端设置为原数据库当中的区块链最后一个区块的哈希值

4. 假如还有没有区块链：

    1. 创建创始区块 // NewGenesisBlock()

    2. 将新建的创始区块存储到数据当中

    3. 将创始区块的哈希保存为最后区块的哈希

    4. 创建一个新的 Blockchain 的实例，然后将其顶端指向创始区块
